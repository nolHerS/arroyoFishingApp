# ArroyoFishingApp

## Descripción
Proyecto backend Spring Boot para gestionar capturas de pesca.

## Tecnologías
- Java 17
- Spring Boot 3.5.6
- MySQL / H2 (para tests)
- Maven
- JaCoCo, SonarCloud
- Swagger/OpenAPI

## Requisitos
- Java 17
- Maven
- MySQL (o H2 en tests)
- Variables de entorno:
    - `DB_PASSWORD`

## Ejecución local
```bash
mvn spring-boot:run
